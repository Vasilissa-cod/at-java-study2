# Домашнее задание 9.1: Ручные тесты для автоматизации

## Тест 1: Расчет ипотеки с валидными данными (аннуитетные платежи)

**Приоритет:** Высокий

### Предусловия:
1. Открыт браузер
2. Переход на сайт https://calcus.ru/kalkulyator-ipoteki
3. Страница полностью загружена

### Шаги выполнения:
1. Перейти на вкладку "По сумме кредита"
2. Ввести сумму кредита: 5 000 000 рублей
3. Ввести срок кредита: 20 лет
4. Ввести процентную ставку: 8.5%
5. Выбрать тип платежей: Аннуитетные (по умолчанию)
6. Нажать кнопку "Рассчитать"
7. Дождаться появления результатов расчета

### Ожидаемый результат:
1. Результаты расчета отображаются на странице
2. Ежемесячный платеж рассчитан и отображается
3. Значение ежемесячного платежа не пустое
4. Дополнительные результаты (начисленные проценты, общая сумма) отображаются

## Тест 3: Расчет ипотеки со сроком в месяцах

**Приоритет:** Средний

### Предусловия:
1. Открыт браузер
2. Переход на сайт https://calcus.ru/kalkulyator-ipoteki
3. Страница полностью загружена

### Шаги выполнения:
1. Перейти на вкладку "По сумме кредита"
2. Ввести сумму кредита: 2 000 000 рублей
3. Ввести срок кредита: 120 месяцев
4. Выбрать единицу измерения: "месяцев" в выпадающем списке
5. Ввести процентную ставку: 9%
6. Выбрать тип платежей: Аннуитетные
7. Нажать кнопку "Рассчитать"
8. Дождаться появления результатов расчета

### Ожидаемый результат:
1. Результаты расчета отображаются на странице
2. Ежемесячный платеж рассчитан и отображается
3. Начисленные проценты рассчитаны и отображаются
4. Общая сумма (долг + проценты) рассчитана и отображается
5. Все значения корректны и не пустые
